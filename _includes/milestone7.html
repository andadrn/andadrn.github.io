MVP value

When finding out the possible users' opinion on how the application should work, we noticed that the general opinion was that the application should have an intuitive interface and it should be easy to use to use such as Uber is. Based on this validation, we decided that our MVP should have the following features: sign up, sign in, post daily jobs(client), accept daily jobs(worker).

Future benefits

We did not manage to implement the rating system that possible users told us it would be great to have, but we included a message in the footer section which informs users that this feature will be available soon.

Feedback loop

We included a button in the footer section so that users can send us an email if they want to send us their feedback or any other questions, as can be seen in the previously uploaded images.
